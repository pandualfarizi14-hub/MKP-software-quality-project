# GRAY BOX TESTING (Pengujian Hybrid)
Gray box testing merupakan kombinasi antara black box dan white box, dengan akses terbatas terhadap struktur internal.

### Teknik-teknik Gray Box Testing

1. Matrix Testing
Menguji hubungan antar variabel:

- Input vs output
- Data mapping

2. Regression Testing

Mengulang pengujian setelah perubahan:

Memastikan tidak ada bug baru

3. Pattern Testing

Menggunakan pola umum:

Berdasarkan histori bug

4. Orthogonal Array Testing

Menggunakan kombinasi variabel secara efisien:

Mengurangi jumlah test case

5. API Testing

Menguji antarmuka sistem:

- Request
- Response
- Validasi data

6. Database Testing

Menguji:

- Struktur database
- Query
- Integritas data

7. Session Testing

Menguji:

- Session management
- Authentication
