# 1. Pendahuluan

## 1.1 Tujuan

Dokumen Software Design Description (DSD) ini bertujuan untuk menjelaskan rancangan teknis sistem aplikasi 
manajemen membership gym berdasarkan kebutuhan yang telah didefinisikan dalam dokumen SRS. Dokumen ini menjadi 
acuan dalam proses implementasi sistem agar pengembangan berjalan terstruktur dan sesuai kebutuhan.

---

## 1.2 Ruang Lingkup

Ruang lingkup perancangan meliputi:

- Desain kebutuhan fungsional sistem
- Desain arsitektur sistem
- Pemodelan sistem menggunakan UML
- Perancangan basis data
- Perancangan antarmuka pengguna

Aplikasi ini mencakup pengelolaan member, jadwal latihan, pembayaran, serta laporan.

---

## 1.3 Gambaran Umum Dokumen

Dokumen Software Design Description (DSD) ini disusun secara sistematis untuk menggambarkan perancangan sistem 
aplikasi manajemen membership gym.

Bagian pendahuluan menjelaskan tujuan, ruang lingkup, dan gambaran umum dokumen. Bagian deskripsi umum menjelaskan 
perspektif produk, manfaat, karakteristik pengguna, serta batasan sistem.

Selanjutnya, bagian software design berisi perancangan sistem yang mencakup kebutuhan fungsional, kebutuhan antarmuka, 
lingkungan operasi, batas perancangan, model sistem menggunakan diagram UML, serta rancangan arsitektur sistem.

Bagian terakhir menjelaskan rancangan antarmuka halaman yang digunakan sebagai gambaran implementasi sistem secara visual.

# 2. Deskripsi Umum
## 2.1 Perspektif Produk
Aplikasi ini merupakan sistem berbasis web yang digunakan untuk mendigitalisasi proses manajemen membership gym, 
termasuk registrasi, pembayaran, dan pengelolaan jadwal latihan.

---

## 2.2 Manfaat Produk

- Mempermudah pengelolaan data member
- Meningkatkan efisiensi transaksi pembayaran
- Menyediakan informasi jadwal latihan secara real-time
- Mengurangi kesalahan pencatatan manual

---

## 2.3 Karakteristik User

| User   | Deskripsi                     |
| ------ | ----------------------------- |
| Admin  | Mengelola seluruh data sistem |
| Member | Menggunakan layanan gym       |

## 2.4 Batasan-Batasan

- Sistem berbasis web
- Membutuhkan koneksi internet
- Menggunakan payment gateway otomatis
- Hak akses dibatasi berdasarkan role (admin & member)

# 3. Software Design

## 3.1 Kebutuhan Fungsional

Fungsi Member
- Registrasi akun
- Verifikasi akun
- Login
- Melihat jadwal latihan
- Melihat status membership
- Melakukan pembayaran
- Perpanjang membership

Fungsi Admin
- Login
- Mengelola data member
- Mengelola jadwal latihan
- Mengelola pembayaran
- Mencetak laporan

---

## 3.2 Kebutuhan Interface

Interface Member
- Halaman registrasi
- Halaman login
- Dashboard member
  - Jadwal latihan
  - Status membership
  - Pembayaran
  - Perpanjangan membership

Interface Admin
- Login admin
- Dashboard admin
  - Kelola member
  - Kelola jadwal
  - Kelola pembayaran
  - Laporan

- Web Server: Apache / Nginx

## 3.6 Rancangan Arsitektur Sistem
Sistem aplikasi manajemen membership gym ini dirancang menggunakan arsitektur Model-View-Controller (MVC) yang bertujuan untuk memisahkan 
antara logika bisnis, tampilan, dan pengelolaan data sehingga sistem lebih terstruktur dan mudah dikembangkan.

Arsitektur sistem terdiri dari beberapa komponen utama, yaitu client, web server, aplikasi, dan database. 
Client merupakan pengguna yang mengakses sistem melalui browser seperti Google Chrome. Permintaan dari client akan 
dikirimkan ke web server untuk diproses oleh aplikasi. Aplikasi kemudian akan berinteraksi dengan database untuk mengambil 
atau menyimpan data yang diperlukan. Hasil proses akan dikembalikan ke client dalam bentuk tampilan antarmuka.

Pada sistem ini, Model berfungsi untuk mengelola data yang tersimpan dalam database seperti data member, jadwal, dan 
pembayaran. View bertanggung jawab dalam menampilkan antarmuka pengguna seperti halaman login, dashboard, dan jadwal latihan.
Controller berfungsi sebagai penghubung antara Model dan View serta mengatur alur logika sistem.

Dengan menggunakan arsitektur MVC, sistem menjadi lebih modular, mudah dalam pemeliharaan, serta mendukung pengembangan 
fitur di masa depan.
