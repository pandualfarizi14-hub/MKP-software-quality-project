# Aplikasi Manajemen Membership Gym

## Deskripsi
Project ini merupakan implementasi Software Quality Assurance (SQA) untuk aplikasi manajemen membership gym berbasis web.

## Tujuan
- Mengelola data membership gym
- Melakukan pengujian software
- Dokumentasi Software Quality
- Implementasi GitHub Collaboration

## Tools
- HTML
- CSS
- PHP
- MySQL
- GitHub

## Jenis Pengujian
- Functional Testing
- Black Box Testing

## Struktur Dokumen
- SKPL
- UML
- SRS
- Test Plan
