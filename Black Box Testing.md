# BLACK BOX TESTING (Pengujian Fungsional)
Black box testing berfokus pada validasi fungsi sistem berdasarkan input dan output, tanpa mengetahui struktur internal program.

### Teknik-teknik Black Box Testing

1. Equivalence Partitioning (EP)

Membagi data input ke dalam kelas-kelas yang ekuivalen:

- Valid class
- Invalid class

2. Boundary Value Analysis (BVA)

Menguji nilai pada batas (boundary):

- Minimum
- Maksimum
- Nilai tepat di sekitar batas

3. Decision Table Testing

Menggunakan tabel keputusan untuk menguji kombinasi kondisi:

- Condition → Action

4. State Transition Testing

Menguji perubahan status sistem:

State awal → event → state berikutnya

5. Use Case Testing

Berdasarkan skenario penggunaan sistem:

Mengikuti alur interaksi user

6. Cause-Effect Graphing

Menggambarkan hubungan sebab-akibat:

- Input (cause)
- Output (effect)

7. Error Guessing

Berdasarkan pengalaman tester:

Menebak kemungkinan error
