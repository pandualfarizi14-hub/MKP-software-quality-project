# WHITE BOX TESTING (Pengujian Struktural)

White box testing berfokus pada logika internal dan struktur kode program.

### Teknik-teknik White Box Testing
1. Statement Coverage

Memastikan semua baris kode dieksekusi minimal satu kali.

2. Branch Coverage

Menguji semua cabang keputusan:

- If-else
- Switch-case

3. Path Coverage

Menguji semua jalur eksekusi program.

4. Condition Coverage

Menguji semua kondisi boolean:

- TRUE
- FALSE

5. Loop Testing

Menguji struktur perulangan:

- Zero iteration
- One iteration
- Multiple iteration

6. Data Flow Testing

Menguji aliran data:

- Definisi variabel
- Penggunaan variabel

Menghindari:

- Variable not initialized
- Unused variable

7. Basis Path Testing

Menggunakan:

- Flow graph
- Cyclomatic complexity
